设备可以使用 WEB 或 CLI 方式登录
CLI 方式是管理员常用的调试方式

VTY 接口

配置级（2级以上）可以完成系统配置
管理级（3级以上）可以对设备进行配置，以及设备升级，故障诊断等
等级越高权限越高（0—3、或者 0—15）

4. 视图之间的切换

不同的视图可以执行不同的命令行信息，对命令进行的分类
用户 > 系统 > 接口、协议、策略等
不同等级的视图需要逐级进入，例如：用户需要通过系统视图才能进入到接口，无法直接从用户视图切换到接口视图
相同等级的视图可以随时切换，例如：从接口视图切换到协议（OSPF）视图
